# Arma 3 Player Tracker (Vue 3 CRUD)

Simple CRUD web app built with Vue 3 using the Composition API and Single File Components (SFC).
Data is stored in an in-memory array (no backend, no persistence).

## Features
- Create: Add a player using a form
- Read: View player list with search + filters
- Update: Edit player details
- Delete: Remove a player

## Tech
- Vue 3
- Vue Router 4
- Vite

## Run locally
```bash
npm install
npm run dev
